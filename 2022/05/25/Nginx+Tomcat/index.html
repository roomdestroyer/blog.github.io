使用 postman 频繁访问，页面出现的频率是 CBCACBCBC CBCACBCBC。

该算法称为提案者轮换选择算法，每一轮都会将被选中的对象减去总权重值，每一轮都会将各结点的权重值加上固定的预设权重值，这是一个公平的轮换算法，可以防止权重过大的结点持续占据资源。过程如下：

初始权重 加权 选择 选中者减去权重和
0 0 0 1 3 5 C 1 3 -4
1 3 -4 2 6 1 B 2 -3 1
2 -3 1 3 0 6 C 3 0 -3
3 0 -3 4 3 2 A -5 3 2
-5 3 2 -4 6 7 C -4 6 -2
-4 6 -2 -3 9 3 B -3 0 3
-3 0 3 -2 3 8 C -2 3 -1
-2 3 -1 -1 6 4 B -1 -3 4
-1 -3 4 0 0 9 C 0 0 0
0 0 0 1 3 5 C 1 3 -4
1 3 -4 2 6 1 B 2 -3 1
2 -3 1 3 0 6 C 3 0 -3
